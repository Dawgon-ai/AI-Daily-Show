# AI Daily Show

An AI-powered news aggregation dashboard that curates and displays the latest AI news from multiple sources.

## Features

- **Real-time AI News**: Aggregates news from multiple AI-focused newsletters and sources
- **Clean Dashboard Interface**: Modern, responsive design for easy browsing
- **Multiple Sources**: Includes content from Hikaru (Ben's Bites), AI Rundown, and other AI news sources
- **Article Images**: Visual previews for each news article
- **Organized by Date**: News articles sorted chronologically for easy navigation

## Project Structure

- `index.html` - Main dashboard interface
- `style.css` - Styling and design system
- `app.js` - Frontend application logic
- `tools/` - Backend scrapers and data processing tools
- `data/` - Stored article data
- `architecture/` - Project architecture documentation
- `Designguidlines/` - Design system and guidelines

## Documentation

- `masterplan.md` - Overall project vision and roadmap
- `task_plan.md` - Current task breakdown and planning
- `findings.md` - Research findings and insights
- `progress.md` - Development progress tracking
- `gemini.md` - AI assistant interaction guidelines

## Getting Started

1. Open `index.html` in a web browser to view the dashboard
2. Run the scrapers in `tools/` to fetch latest articles
3. Data is stored in the `data/` directory

## Technologies

- HTML5, CSS3, JavaScript
- Python (for web scraping tools)
- Modern web design principles
